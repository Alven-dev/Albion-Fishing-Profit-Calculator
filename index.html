Albion Online Fish Profit calculator

How does it work?

  1. Click the button below to set the Chopped Meat current market price or estimated value (it's up to you!).
  2. Select the fish you'd like to check.
  3. Set the current market value or estimated value of the selected fish.
  4. Click the Check button to verify profit and whether it's better to sell the fish or process it.
  5. After each use you'll have to use "Reset" button, in order to use the calculator again.
